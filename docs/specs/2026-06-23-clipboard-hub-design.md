## 一、概述

Clipboard Hub 是一个 Windows 桌面剪贴板历史管理器。自动记录每次复制的内容，用户可随时查看历史、搜索、一键重新复制。采用「热角抽屉」交互——紧贴桌面右上角的半透明玻璃条，鼠标移入展开完整面板，移开面板自动收起（300ms 延迟）。

### 四大核心模块

| 模块 | 职责 | 技术实现 |
|------|------|---------|
| `ClipboardWatcher` | 每 500ms 轮询系统剪贴板，检测新内容写入 Store，识别自身写入避免循环 | `QApplication.clipboard()` + `QTimer` + MD5 哈希去重 |
| `ClipboardStore` | 内存列表，增/删/查/重拷，去重，信号通知 UI | `QObject` + `Signal` |
| `TabBar` | 紧贴屏幕右上角的无文字玻璃指示条，检测鼠标进入/离开，发射面板显隐信号，支持拖动和自动吸附 | `Qt.WA_TranslucentBackground` + `paintEvent` 渐变 + `QPropertyAnimation` 吸附 |
| `HistoryPanel` | 搜索框 + 历史列表 + 逐条删除按钮，点击条目重新复制（不自动隐藏） | 自定义 `ItemRow` 每项独立 `paintEvent` |

## 三、组件设计

### 3.1 ClipboardStore（数据层）

#### 数据结构

```python
{
    "id": "uuid4",             # 唯一标识
    "type": "text" | "image" | "html",
    "content": "...",          # 文本/HTML 原文；图片为 base64 字符串
    "preview": "...",          # 前 80 字符截断预览；图片为 "图片 (W×H, 大小)"
    "timestamp": 1702700000,   # Unix 时间戳
    "app_source": "unknown"    # 来源应用（尽力获取）
}
```

#### 接口

| 方法 | 触发 | 行为 |
|------|------|------|
| `add(data)` | Watcher 检测到新内容 | 去重（与最新一条内容全等比较），头插，发 `items_changed` 信号 |
| `delete(item_id)` | 用户确认删除 | 移除，发 `items_changed` |
| `get_all()` | 面板初始加载 | 返回全部列表（按时间倒序） |
| `search(keyword)` | 搜索框输入 | 返回 `content`/`preview` 含关键词的条目（大小写不敏感） |
| `get_by_id(item_id)` | 用户点击条目 | 按 ID 查找并返回 ClipItem，找不到返回 None |
| `re_copy(item_id)` | (兼容别名) | 同 get_by_id |

#### 自复制防止机制

用户点击条目重新复制时：
1. `HistoryPanel._on_re_copy` 计算内容 MD5
2. 调用 `Watcher.notify_self_copy(md5)` 设置 `_is_self_copy=True` 并更新 `_last_md5`
3. 下次 `_poll` 检测到相同 MD5 → 跳过，不重复记录

#### 去重规则

- 文本/HTML：MD5 哈希比较
- 图片：像素数据 MD5 比较
- 连续复制相同内容 → 不重复添加

### 3.2 TabBar（悬浮玻璃指示条）

**设计理念：** 极简无文字，像桌面的一部分。白色半透明玻璃材质，渐变边框在浅色/深色桌面上都能看到。

```
┌──────────────────────────────────────────┐
│                                          │ ← 16px 高，紧贴屏幕右上角
└──────────────────────────────────────────┘
  (无文字，纯玻璃条，渐变边框)
```

- 尺寸：**380×16 px**（与面板等宽）
- 位置：**紧贴屏幕右上角**（`x = screen.right - width`, `y = screen.top`），零空隙
- 属性：无边框、置顶 (`Qt.WindowStaysOnTopHint`)、无任务栏图标 (`Qt.Tool`)、透明背景 (`WA_TranslucentBackground`)
- **玻璃材质：** 通过 `paintEvent` 自定义绘制
  - 背景：`rgba(255, 255, 255, 70)` 白色半透明
  - 渐变边框：`QLinearGradient` 从上到下透明度递增
    - 0%: `rgba(180, 180, 190, 60)`
    - 50%: `rgba(150, 150, 160, 110)`
    - 100%: `rgba(120, 120, 130, 160)`
  - 圆角：4px
- **拖动：** 按住左键拖到任意位置，面板实时跟随（通过 `position_changed` 信号）
- **自动吸附：** `mouseReleaseEvent` 启动 600ms 延时 → `QPropertyAnimation` 250ms 平滑动画回到屏幕右上角
- 信号：
  - `panel_show_requested()` — 鼠标进入
  - `panel_hide_requested()` — 鼠标离开（面板有 300ms 延迟隐藏）
  - `position_changed(x, y)` — 拖动时通知面板跟随

### 3.3 HistoryPanel（玻璃风格展开面板）

```
┌─────────────────────────────────────────┐
│  🔍  搜索历史...                         │  ← SearchBox (玻璃边框)
├─────────────────────────────────────────┤
│  ┌─────────────────────────────────┐ ✕  │
│  │ ● 复制的文本内容（最多两行...）  │    │  ← 条目卡片 (独立玻璃+border)
│  └─────────────────────────────────┘     │
│  ┌─────────────────────────────────┐ ✕  │
│  │ ● 🖼 图片 (1920×1080, 245KB)   │    │
│  └─────────────────────────────────┘     │
│  ┌─────────────────────────────────┐ ✕  │
│  │ ● <div class="container">...    │    │
│  └─────────────────────────────────┘     │
│                                          │
│           ... 滚动查看更多 ...            │
└─────────────────────────────────────────┘
```

- 尺寸：**380×480 px**，与 TabBar 等宽
- 位置：TabBar 正下方，拖动 TabBar 时面板实时跟随
- 属性：无边框、置顶、无任务栏图标、`WA_TranslucentBackground`
- **面板玻璃材质：**
  - 背景：`rgba(250, 250, 252, 60)` 白色基调半透明
  - 渐变边框 1px：
    - 0%: `rgba(170, 170, 180, 80)`
    - 30%: `rgba(150, 150, 160, 130)`
    - 70%: `rgba(130, 130, 140, 160)`
    - 100%: `rgba(110, 110, 120, 190)`
  - 圆角：12px

#### 搜索框

- 背景：`rgba(255, 255, 255, 55)`
- 边框：`1.5px solid rgba(140, 140, 150, 160)` 可见边框
- 聚焦边框变蓝：`#89B4FA`
- 圆角 8px
- 文字颜色：`rgba(40, 40, 45, 240)` 深色，浅/深桌面都可见
- 输入即过滤，无匹配时显示 "无匹配记录"

#### 条目卡片（HistoryList ItemRow）

每条历史记录是一个独立卡片，通过自定义 `ItemRow.paintEvent` 绘制：

- 背景：`rgba(255, 255, 255, 50)` — 比面板背景略深
- 渐变边框 0.5px：
  - 0%: `rgba(160, 160, 170, 70)`
  - 100%: `rgba(120, 120, 130, 140)`
- 圆角：8px
- 布局：`QLabel`（预览文本） + `QPushButton`（✕ 删除按钮）
- 预览文本：
  - 字体：JetBrains Mono 10pt
  - 颜色：`rgba(40, 40, 45, 240)` 深色
  - 自动换行，最大高度 36px（约两行）
  - `Qt.UserRole` 存储 `item_id`
- 删除按钮 ✕：
  - 平时：`rgba(60, 60, 65, 160)`
  - 悬浮：`#F38BA8`
  - 点击 → 弹出确认框 → 确认后删除
- 类型圆点：文本 `#7AA2F7` | 图片 `#89B4FA` | HTML `#FAB387`

| 操作 | 效果 |
|------|------|
| 鼠标移入 TabBar | 面板 `show_at()` 在 TabBar 正下方 |
| 鼠标在 TabBar 或 Panel 内 | 保持展开 |

## 四、数据流

```
系统剪贴板变化
      │
      ▼
ClipboardWatcher (QTimer 每 500ms 轮询)
      │  检测到新内容（文本/图片/HTML）
      │  MD5 去重（与上次内容比较）
      │  识别 self_copy 标记 → 跳过
      ▼
ClipboardStore.add()  ──→  items_changed 信号
      │                        │
      │                        ▼
      │                   HistoryPanel
      │                   刷新列表 + 重新过滤
      │
      │  用户点击条目
      ▼
ClipboardStore.get_by_id() ──→ 计算 content MD5
      │                        │
      │                        ▼
      │                   Watcher.notify_self_copy(md5)
      │                   设置 _last_md5，防止循环记录
      │                        │
      ▼                        ▼
QApplication.clipboard().setText/setPixmap()
(面板不隐藏，等鼠标移出)
```

### 交互流

```
[鼠标移入 TabBar] → HistoryPanel.show_at() 在 TabBar 正下方
[鼠标在 TabBar 或 Panel 内] → 保持展开（取消 hide_timer）
[鼠标移出两者] → 300ms 延迟 → HistoryPanel.hide()
[点击条目] → get_by_id → 写回剪贴板 → notify_self_copy(md5) → 面板不隐藏
[点击 ✕] → QMessageBox.question 确认 → 是: Store.delete()
[搜索框输入] → Store.search(keyword) → 列表实时过滤
[拖动 TabBar] → position_changed 信号 → Panel 实时跟随
[松开 TabBar] → 600ms 后 QPropertyAnimation 自动吸附右上角
```

## 六、错误处理 & 边界情况

### 边界情况

| 场景 | 处理 |
|------|------|
| 剪贴板为空 | 跳过，不添加空条目 |
| 超长文本 (>10000 字符) | 预览截断 80 字符，完整保留在内存 |
| 超大图片 (>10MB) | 跳过不记录 |
| 连续复制同一内容 | MD5 去重，不做任何操作 |
| 剪贴板来自本程序 re-copy | `notify_self_copy(md5)` + MD5 匹配 → 跳过 |
| 面板展开时其他窗口抢占焦点 | 不影响，面板保持置顶 |
| 屏幕分辨率/多显示器变化 | TabBar 自动吸附回到主屏右上角 |

### 错误处理

| 错误 | 处理 |
|------|------|
| 剪贴板访问失败 | 静默跳过（RuntimeError/OSError/AttributeError），下次重试 |
| HTML 格式获取失败 | 降级为纯文本 |
| 图片格式获取失败 | 跳过图片，保留文本版本（如有） |
| re-copy 写回失败 | 静默，不崩溃 |

### 不处理（YAGNI）

- 文件复制（CF_HDROP）
- 跨设备同步
- 数据加密
- 持久化存储
- 全局快捷键
- 清空全部历史功能
